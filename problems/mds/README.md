## Idea clave

QAP es un problema de **minimizacion** sobre dos matrices (flujos y distancias).
El coste de una solucion es el doble sumatorio cuadratico entre ambas, por eso
es buen benchmark combinatorio cuando se hacen muchas evaluaciones.
